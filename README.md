# BetaDataSDK

BetaData是数据统计的SDK

## 案例

运行这个工程, 先 clone the repo, 之后在工程根目录下运行  `pod install` 

## 依赖
* iOS8以上


## 安装

BetaDataSDK可以通过[CocoaPods](https://cocoapods.org)获得。安装
它，只需添加以下行到您的Podfile:

```ruby
pod 'BetaDataSDK'
```
